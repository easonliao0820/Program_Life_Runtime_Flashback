📖 程式人生：跑馬燈 (Program_Life_Runtime_Flashback)
「如果人生是一段程式碼，你會如何編寫你的回憶？」

《程式人生：跑馬燈》是一款結合 Python 語法教學 與 台灣文化生活 的敘事型 RPG 遊戲。玩家將扮演主角「小艾」，在神秘 AI 導師「派森」的引導下，從嬰兒、幼兒、國中到大學，透過解決生活中的程式難題，重新體驗並編寫自己的人生片段。

🌟 核心特色
沉浸式教育：將 Python 基礎語法（變數、運算、邏輯判斷）融入生活情境（如：路邊早餐店、便利商店）。
